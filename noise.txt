================================================================================
  TROUBLESHOOTING GUIDE
================================================================================

PROBLEM 1: "python: command not found" or similar error
SOLUTION:
  ‚Ä¢ Python might not be in your PATH
  ‚Ä¢ Try: py main.py instead of python main.py
  ‚Ä¢ Or check Python installation


PROBLEM 2: Webcam doesn't open
SOLUTION:
  ‚Ä¢ Check if webcam is connected and working
  ‚Ä¢ Close other apps using webcam (Zoom, Teams, Skype)
  ‚Ä¢ Try running: python test_system.py to diagnose
  ‚Ä¢ Check if webcam light turns on


PROBLEM 3: Hand not detected (no green lines)
SOLUTION:
  ‚Ä¢ Improve lighting - turn on more lights
  ‚Ä¢ Move closer to camera (1-2 feet distance)
  ‚Ä¢ Make sure hand is in center of camera view
  ‚Ä¢ Keep hand still for a moment
  ‚Ä¢ Avoid backlighting (don't sit with bright window behind you)


PROBLEM 4: Wrong gesture detected
SOLUTION:
  ‚Ä¢ Make clearer, more exaggerated gestures
  ‚Ä¢ Hold gesture steady for at least 1 second
  ‚Ä¢ Your datasets have only 2 samples each - consider collecting more
  ‚Ä¢ Run: python collect_gestures.py to add more samples


PROBLEM 5: Gestures not controlling the video
SOLUTION:
  ‚Ä¢ Make sure video window is ACTIVE (click on it)
  ‚Ä¢ Keyboard shortcuts only work on the focused window
  ‚Ä¢ Try clicking on the video player again
  ‚Ä¢ Make sure video player supports keyboard shortcuts


PROBLEM 6: Actions happen too fast or too slow
SOLUTION:
  ‚Ä¢ System has 0.8 second cooldown between actions
  ‚Ä¢ Hold gestures for full 1 second
  ‚Ä¢ Don't switch gestures too quickly
  ‚Ä¢ If you want to change cooldown, edit main.py line 47


PROBLEM 7: "Could not convert string to float" error
SOLUTION:
  ‚Ä¢ Your data was already fixed!
  ‚Ä¢ If you see this, run: python fix_data_format.py
  ‚Ä¢ This will reformat the gesture data files


PROBLEM 8: Some video players don't respond
SOLUTION:
  Different players use different shortcuts:
  
  YouTube: All shortcuts work perfectly ‚úì
  Netflix: Spacebar and F work ‚úì
  VLC: All shortcuts work ‚úì
  Windows Media Player: All shortcuts work ‚úì
  
  If volume doesn't change, use on-screen controls

================================================================================
  FREQUENTLY ASKED QUESTIONS
================================================================================

Q: Do I need internet?
A: Only if you want to control online videos (YouTube, Netflix).
   For local videos (VLC, Media Player), no internet needed.

Q: Can I use this while eating?
A: Yes! That's one of the main use cases. Control videos hands-free!

Q: Will it work in the dark?
A: No, hand detection needs good lighting. Turn on room lights.

Q: Can I customize which gesture does what?
A: Yes! Edit main.py file, lines 38-43 to change gesture actions.

Q: How accurate is it?
A: With 2 samples per gesture: Good (70-80% accurate)
   With 30+ samples per gesture: Excellent (95%+ accurate)

Q: Does it work with PowerPoint?
A: Yes! If PowerPoint is in presentation mode and active.

Q: Can I control volume of my speakers?
A: It sends volume commands to the video player, not system volume.

Q: Why do I need to click on the video first?
A: Keyboard commands only work on the "active" (focused) window.

Q: Can I minimize the webcam window?
A: No, you need to see it. But you can move it to a corner.
